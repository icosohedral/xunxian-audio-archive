# 寻仙音乐资料库

一个整理、检索与试听《寻仙》游戏音乐和音效的非官方资料库。

项目希望把散落在游戏文件中的声音重新编目，让熟悉的城镇旋律、山野环境声与战斗音效都能更容易地被找到、听见和辨认。

> 让每一段声音，都能被重新听见。

## 在线访问

[music.xunxian.wiki](https://music.xunxian.wiki)

## 收录内容

资料库目前分为两个部分：

- **仙乐集**：收录 148 首游戏背景音乐，可按场景与类别浏览、搜索和连续播放。
- **万籁库**：收录 6,266 条游戏音效，涵盖环境、角色、技能、战斗等声音，可分类筛选并单独试听。

曲目和音效保留原始文件名称，同时通过标签和正则分类规则建立更直观的索引。分类规则支持导入与导出，方便持续整理和维护。

## 页面一览

- **总览**：了解资料库的收录范围与主要内容。
- **仙乐集**：寻找和播放《寻仙》中的背景音乐。
- **万籁库**：检索数量庞大的游戏音效。
- **关于**：了解项目缘起、资料边界与使用说明。

## 项目说明

本项目由爱好者整理，用于游戏音乐与音效的资料研究、检索和个人欣赏，并非《寻仙》官方网站。

仓库不会提交原始音频或生成的试听文件。公开目录中也不会包含本地文件的绝对路径。音频版权归其原权利人所有，请勿将资料用于商业用途。

## 本地运行

项目使用 pnpm 管理。拥有本地资料源时，可以运行：

```bash
pnpm install
pnpm catalog:build
pnpm dev
```

随后访问 <http://localhost:3000>。

`original_data/` 是只读的本地资料目录，不包含在公开仓库中，项目不会修改其中的文件。
